Discover Learning Resources

This is a collection of hands-on learning activities developed by Oxford University staff, demonstrating ways to incorporate digital tools into teaching. Explore the collection by entering a search term, selecting a category, viewing a recently added item or clicking the 'See more...' link below. See individual activities for information on how to gain permission to re-use them as-is, or use them to learn a new skill or be inspired to adopt a new digital tool yourself.
